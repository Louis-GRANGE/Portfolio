Solar System

"Solar System" is a procedural generation project that allows users to create their own solar systems. By adjusting various parameters, users can generate asteroids, alter planetary orbits, and customize the shape of planets. The project focuses on providing an interactive and dynamic way to visualize solar system creation in Unity 3D.
