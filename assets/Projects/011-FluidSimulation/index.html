Real-Time Fluid Simulation

This project is a real-time fluid simulation developed as part of a Bachelor's program in 3D Interactive Graphic Applications. It demonstrates a physically-based simulation of fluid behavior using Unity, focusing on both realistic movement and visual rendering. The simulation applies Navier-Stokes equations to reproduce fluid dynamics within an interactive 3D environment.
