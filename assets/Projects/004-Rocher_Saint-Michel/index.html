Rocher Saint-Michel

"Rocher Saint-Michel" is a software created using Unreal Engine 4, allowing the user to explore the iconic Rocher Saint-Michel located in Le Puy-en-Velay (43). The character can freely navigate the environment, change the surroundings, and add 3D models. Additionally, the user can access key points of interest within the historic monument, creating an immersive experience.

My Duties:

  • Programmed the entire user interface (UI) to enable seamless interaction with the environment.
  • Implemented interactions with 3D models to allow for dynamic changes in the scene.
