Files Converter & TextToSpeech UnrealEngine Plugin

This plugin is used to convert multiple file types using the free API: API2Convert (for example, converting MP3 to WAV) & also includes text-to-speech functionality via the free API: Voice RSS, directly integrated in Unreal Engine's Widget Editor.
