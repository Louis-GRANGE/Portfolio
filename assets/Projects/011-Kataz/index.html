Kataz

Kataz is an interactive game developed using Microsoft Azure Cognitive Services. The game showcases the integration of various AI services, including Speech to Text, LUIS (Language Understanding), and Computer Vision, within a Unity WebGL environment. The goal is to demonstrate the potential of Azure's AI capabilities through an engaging gameplay experience.
